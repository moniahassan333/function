ميزات الأمان في المشروع

  • حماية من SQL Injection باستخدام PDO و Prepared Statements
  • حماية من XSS بتنظيف جميع المدخلات
  • حظر بعد 3 محاولات فاشلة لتسجيل الدخول وإعادة فتح بعد 3 دقائق
  • تشفير كلمات المرور باستخدام password_hash
  • حماية CRUD و التأكد من ملكية المستخدم للمهمة
  • CSRF Tokens لكل عملية تعديل أو حذف أو إضافة
  • جلسات آمنة مع session_regenerate_id و httponly و samesite
  • إشعارات ديناميكية للنجاح أو الأخطاء تظهر داخل الواجهة (toast)
  • واجهة RTL منظمة وأنيقة لتسهيل الاستخدام
